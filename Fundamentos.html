Anatomía de un elemento HTML

Un elemento HTML generalmente consta de una etiqueta de apertura, contenido y una etiqueta de cierre.

<h1>Título principal</h1>

en este caso, <h1> es la etiqueta de apertura, Título principal es el contenido y </h1> es la etiqueta de cierre.

Contenido de un elemento HTML

Un elemento HTML generalmente consta de una etiqueta de apertura, contenido y una etiqueta de cierre.

<h1>Título principal</h1>

en este caso, <h1> es la etiqueta de apertura, Título principal es el contenido y </h1> es la etiqueta de cierre.

Los atributos proporcionan información adicional sobre un elemento HTML. Ejemplo: class="nota"
